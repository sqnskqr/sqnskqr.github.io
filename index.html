Technical Projects

Project Pacman - Python

Project Pacman was an amazing introduction to optimizing a zero-sum game of Pacman vs Ghosts. The project explored adversarial agents, heuristic search, alpha-beta minimax optimization, and reinforcment learning. This allowed for an end product that determined optimal movements for the Pacman agent against multiple optimally performing adversarial agents.

python, AI/ML, optimization, pathfinding, data structures and algorithms

Build Your Own World - Java

This project was a brief fullfillment of a classic childhood dream in creating my own expansive video game world. I implemented a graphical interface and pathfinding algorithms such as A* to create interactive, seed-based, randomly generating 2D worlds.

java, pathfinding, data structures and algorithms, user interface

Gitlet - Java

This tedious yet enjoyable project essenstially entailed developing a version-control system similar to Github. Gilet is capable of creating and editing multiple remote repositories through sha-1 file identification and serialization that enabled persistence.

java, data structures and algorithms, version-control system

Scheme Interpreter - Python

An interpreter for Scheme, a Lisp dialect programming language, was a multifaceted project. The first step was understanding the syntax of a functional programming language. After fully learning how to use the language, I was able to use higher order funtions, object-oriented programming, string parsing, and allowing for interactive user input.

python, compiling, user interface
